# Flutter To-Do List UI App

A simple and responsive Flutter To-Do List UI application built using basic Flutter widgets and layout system.

This project demonstrates:
- Scaffold & AppBar
- Task List UI using Cards
- Add Task Screen UI
- Navigation between screens
- ListView layout system
- Clean and beginner-friendly Flutter structure

---

## 📱 Features

✅ Beautiful To-Do List UI  
✅ Task Cards with Status & Time  
✅ Add Task Screen  
✅ Screen Navigation using Navigator  
✅ Responsive UI Design  
✅ Clean and Structured Code  

---

## 🛠️ Technologies Used

- Flutter
- Dart

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/flutter-todo-list-ui.git
```

### 2. Navigate to project folder
```bash
cd flutter-todo-list-ui
```

### 3. Install dependencies
```bash
flutter pub get
```

### 4. Run the app
```bash
flutter run
```
